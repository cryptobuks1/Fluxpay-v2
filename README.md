

# Fluxpay v2

## Requirements

Make sure you have the following:
		<ol>
			<li>PHP >= 7.0.0</li>
			<li>PDO PHP Extension</li>
			<li>Tokenizer PHP Extension</li>
			<li>OpenSSL PHP Extension</li>
			<li>XML PHP Extension</li>
			<li>Mbstring PHP Extension</li>
			<li>GD Library</li>
			<li>CURL Library</li>
			<li>PHP mod_rewrite enabled</li>
		</ol>
